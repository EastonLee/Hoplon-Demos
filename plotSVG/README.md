# plotSVG

A Hoplon demo that plots some data using SVG.

![example plot][2]

[Demo][3]

## Dependencies

- java 1.7+
- [boot][1]

## Usage

1. Start the auto-compiler. In a terminal:

    ```bash
    $ boot development
    ```

2. Go to [http://localhost:8000][4] in your browser.

3. You're already done.
